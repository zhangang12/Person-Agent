## 0. 一句话

记忆系统的目标不是"记住一切",而是**下次少探索一遍**。对话转录不等于记忆;只有过了判据、挂了锚点、能机械核验的事实才配进长期库。

## 1. 分层模型:四种"记忆",四种生命周期,绝不混写

| 层 | 存储 | 内容 | 谁写 | 作用域 | 注入策略 |
|---|---|---|---|---|---|
| L0 个人记忆 | `userData/memory.md` | 偏好/习惯/角色("我是信贷后端") | **用户写**,Agent 只建议不直写 | 全局 | 每卡首条必带,≤2k |
| L1 项目知识库 | `userData/knowledge/<项目名_短hash>.md` | 系统级事实(结构/规则/部署真相) | **Agent 蒸馏写**(`memory_add`,三问判据+壳层校验) | 按项目分库 | 场景命中优先+新→旧,≤6k/60 条 |
| L2 项目自述 | 仓库内 `CLAUDE.md/README.md` | 团队共识 | 仓库维护 | 项目 | 只读引用,≤5.5k |
| L3 会话接力 | 内存(压缩摘要/工作流注册表快照) | "这事做到哪了" | 壳层自动 | 单卡单次 | **不倒灌长期记忆** |

要点:

- **L3 不进长期库**(设计备忘 §8:探索烧在一次性子上下文,主线程只收蒸馏结论)。工作流存档 `userData/workflows/` 也属 L3,供回取不供注入。
- **L0 是用户主权**:Agent 不得直写;若 Agent 想建议记忆,走对话让用户自己改。
- **L1/L2 按项目隔离**:同名项目靠目录 hash 区分;不跨项目共享(污染)。

## 2. 写入管道(蒸馏闭环)

```
任务收尾(工作流规程第10条,后续扩展到编排/长对话)
  → Agent 自评三问判据(① 系统级? ② 三个月后仍真? ③ 有重用场景?)
  → memory_add(text + anchors + scene + confidence)
  → 壳层硬校验(确定性,不信模型自证)
      · 锚点必填(无源不立):C1 文件存在? C2 符号 grep 到?
      · C1 失败 → 拒写;C2 失败 → 拒写或降级 suspected
  → 追加去重(完全匹配,近似合并不做)
  → 审计留痕(audit.jsonl)
```

- 条目格式(纯 markdown,锚点用相对路径,**与仓库可平移的兼容现在锁定**):
  `- [verified] 计息规则在 InterestCalc.monthly(),按月复利 (锚点: src/interest/calc.js:88) (场景: 计息/跑批)`
- 判据写进工具描述与规程双份(模型看得见的地方才有效);没够格的事实**宁缺勿滥**。

## 3. 读出管道(注入)

- **预算制**:首条消息背景总预算 ~12k 字符;L0≤2k / L1≤6k / L2≤5.5k,超出在头部明示略去条数(不让模型误以为看到全部)。
- **防腐校验(注入前做,已实现:`src/knowledge.js` `auditEntries()`)** —— 过时地图比没地图危险:
  - C1 文件没了 / C2 符号没了 → 该锚点死;**全死 → 该条隔离不注入(标红),部分死 → 标黄仍注入**;
  - C3 行漂移 → 就近重定位(离旧行号最近的符号出现行),**回写知识库文件里的新锚点行号**,条目照常注入;
  - C4 自 verified(条目无独立字段,用所在 `## 日期` 节做代理)以来锚点文件 churn 超阈(默认 300 行,参数 `churnMaxLines`)→ 标黄,注入带 `[待复核]` 标记;
  - 健康度结果(`{line, status: green|yellow|red, reasons[]}`)供治理界面直接复用;文件/churn 检查结果按 mtime/日期做进程内缓存,开卡热路径不全量 grep/git。
- **两级索引(已实现:`injectText()` 的 `target` 参数)**:
  - 一级:按 scene 分词/锚点路径与文件名对当前目标文本(卡片标题/goal 片段/接力摘要)做**确定性命中**(大小写不敏感子串),命中必注入、优先占预算;
  - 二级:其余条目新→旧兜底;预算仍是 ≤60 条/≤6000 字,命中超预算截断并在头部明示;
  - embedding 等条目上千再说,现在上是过度工程。

## 4. 治理(待实现)

- **可视化**:卡坞/设置里按项目看条目 + 健康度(绿/黄/红),可编辑删除(现在只能翻文件)。
- **衰减**:锚点全死 + 长期未命中 → 进隔离区(不删,可恢复)。
- **旋钮化**:全部阈值(注入预算、压缩 80%、自动压缩 5 次、todo 提醒 3 轮、churn 阈值)进 `settings.json` —— 弱模型补偿做成可拧松的旋钮,模型升级后一档档退掉(备忘 §4.6);知识资产本身是永久的。

## 5. 存储决策(已定)

- **壳层私有存储**(`userData/`),对仓库零侵入 —— 写进团队 `docs/` 需要团队认可,个人智能体定位优先(见 `docs/项目记忆/project_personal_agent_focus.md`)。
- 迁移路径:条目就是纯 markdown + 相对路径锚点,团队接受后可原样平移进仓库;决策可推迟,**格式兼容必须现在锁定**(备忘 §7.2)。

- 不把对话转录/工作流存档当记忆注入(L3 永不倒灌)
